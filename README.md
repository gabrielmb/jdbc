# JDBC

<strong>Ferramentas:</strong>
<ul>
  <li>Java 8</li>
  <li>MySQL - Ver 5.7.x</li>
  <li>IDE Eclipse</li>
  <li>SGBD - Workbench 6.3 CE</li>
</ul>

<hr>

A finalidade deste código e prover um entendimento básico de como se comunicar com um banco de dados utilizando a API JDBC.
